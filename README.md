## Project Overview:

Coin paprika api를 이용한 Coin Tracking application.

## What I did:

- API로부터 코인 데이터를 실시간으로 받아와 클라이언트에게 보여주기
- Apexcharts를 이용하여 1주~6달까지 기간별 차트 보여주기
- Infinite scroll
- 검색기능

## Conclusion:

- React query를 통한 데이터 캐싱과 코드의 간소화
